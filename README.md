# User Management System (Spring Boot + JWT Security)

This is a Spring Boot User Management project with **Spring Security + JWT Authentication**.

## Features
- User CRUD Operations
- Spring Boot REST APIs
- MySQL Database Integration
- Lombok for reducing boilerplate code
- Spring Security with JWT Token Validation
- Secured endpoints (Authorization required)

---

## Tech Stack
- Java
- Spring Boot
- Spring Data JPA (Hibernate)
- Spring Security
- JWT (JSON Web Token)
- Lombok
- MySQL
- Maven
